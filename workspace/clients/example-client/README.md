# Example Client

Placeholder client folder showing the campaign convention. One folder per real client lives here. Inside each, a `campaigns/` directory holds one folder per campaign.

```
example-client/
└── campaigns/
    └── saas-founders-2026-q2/
        ├── campaign-strategy.md
        ├── leads.csv          (created by find-leads)
        ├── copy.md
        ├── campaign-status.md (created by create-campaign)
        └── campaign-analysis.md (filled after results)
```

Delete this client folder when you start adding real ones, or rename it to your first client.
